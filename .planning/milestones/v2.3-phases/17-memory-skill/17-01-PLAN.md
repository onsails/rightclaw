    **2. Create `crates/rightclaw/src/memory/store.rs`:**

    Define return struct:
    ```rust
    #[derive(Debug, Clone)]
    pub struct MemoryEntry {
        pub id: i64,
        pub content: String,
        pub tags: Option<String>,
        pub stored_by: Option<String>,
        pub source_tool: Option<String>,
        pub created_at: String,
        pub importance: f64,
    }
    ```

    Implement four public functions, all taking `&rusqlite::Connection` as first arg:

    **`store_memory(conn, content, tags, stored_by, source_tool) -> Result<i64, MemoryError>`:**
    - Call `super::guard::has_injection(content)` — if true, return `Err(MemoryError::InjectionDetected)`
    - `INSERT INTO memories (content, tags, stored_by, source_tool) VALUES (?1, ?2, ?3, ?4)`
    - Get `last_insert_rowid()` as the new memory ID
    - `INSERT INTO memory_events (memory_id, event_type, actor) VALUES (?1, 'store', ?2)` with memory_id and stored_by
    - Return `Ok(id)`

    **`recall_memories(conn, query) -> Result<Vec<MemoryEntry>, MemoryError>`:**
    - `SELECT id, content, tags, stored_by, source_tool, created_at, importance FROM memories WHERE deleted_at IS NULL AND (tags LIKE '%' || ?1 || '%' OR content LIKE '%' || ?1 || '%') ORDER BY created_at DESC LIMIT 50`
    - Map rows to `MemoryEntry` vec
    - Return `Ok(entries)`

    **`search_memories(conn, query) -> Result<Vec<MemoryEntry>, MemoryError>`:**
    - `SELECT m.id, m.content, m.tags, m.stored_by, m.source_tool, m.created_at, m.importance FROM memories m JOIN memories_fts f ON m.id = f.rowid WHERE memories_fts MATCH ?1 AND m.deleted_at IS NULL ORDER BY bm25(memories_fts) LIMIT 50`
    - Map rows to `MemoryEntry` vec
    - Return `Ok(entries)` (empty vec if FTS5 match finds nothing)

    **`forget_memory(conn, id, actor) -> Result<(), MemoryError>`:**
    - First verify the row exists and is not already deleted: `SELECT id FROM memories WHERE id = ?1 AND deleted_at IS NULL`
    - If not found: return `Err(MemoryError::NotFound(id))`
    - `UPDATE memories SET deleted_at = datetime('now') WHERE id = ?1`
    - `INSERT INTO memory_events (memory_id, event_type, actor) VALUES (?1, 'forget', ?2)`
    - Return `Ok(())`
