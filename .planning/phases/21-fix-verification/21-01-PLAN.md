<objective>
Fix the rightcron bootstrap failure that prevents `*/5 * * * *` reconciler job creation.

Purpose: CC's CronCreate tool is unavailable in subagent contexts. The current startup_prompt
instructs CC to use the Agent tool (spawns a subagent), which cannot access CronCreate. Removing
the Agent tool delegation runs rightcron in the main thread where CronCreate IS available.
This creates the reconciler job, which fires every 5 minutes, calling M6() and draining
the Telegram message queue post-SubagentStop.

Output:
- Updated startup_prompt in shell_wrapper.rs (single string change)
- New failing test added FIRST (TDD), then string fixed to make it pass
- All existing tests remain green
</objective>
