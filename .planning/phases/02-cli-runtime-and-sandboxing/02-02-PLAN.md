<!-- process-compose REST API endpoints from research -->
Endpoints:
- GET /processes -> list all processes with status
- GET /process/{name} -> single process info
- POST /process/restart/{name} -> restart one process
- PATCH /process/stop/{name} -> stop one process
- POST /project/stop -> stop all (shutdown)
- GET /live -> health check

<!-- reqwest Unix socket pattern from research -->
```rust
let client = Client::builder()
    .unix_socket(socket_path)
    .build()?;
// base_url can be "http://localhost" -- host ignored for UDS
```
