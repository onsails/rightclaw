| Path traversal via chrome_path or mcp_binary_path in config.yaml | Fields stored as PathBuf (no shell expansion); values passed as JSON array args in Plan 02 (not shell string interpolation) | medium |
| Malformed config.yaml causes panic | serde-saphyr is panic-free on malformed input; read_global_config returns Err not panics; existing pattern verified | low |
| Secrets in chrome_path value (e.g. embedded credentials) | Values written to per-agent .mcp.json command field — paths only, no secrets expected; standard filesystem path format | low |
</threat_model>
