## Decisions Made
- **rustls over native-tls:** Plan specified `native-tls`, but the devenv (Nix) does not expose OpenSSL via pkg-config. openssl-sys build script fails with "Could not find directory of OpenSSL installation". `rustls` is pure-Rust, works without system libraries, and is already the TLS backend for the workspace `reqwest` dependency. This makes the workspace consistently rustls-only.
