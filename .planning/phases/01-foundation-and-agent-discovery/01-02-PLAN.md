    2. **crates/rightclaw/src/agent/discovery.rs**:

       `validate_agent_name(name: &str) -> Result<(), AgentError>`:
       - Check name matches regex pattern: `^[a-zA-Z0-9][a-zA-Z0-9_-]*$`
       - Use a manual char check (no regex crate needed for this simple pattern): name must be non-empty, first char alphanumeric, all chars must be alphanumeric, hyphen, or underscore
       - Return `AgentError::InvalidName` on failure

       `parse_agent_config(agent_dir: &Path) -> miette::Result<Option<AgentConfig>>`:
       - Check if `agent_dir.join("agent.yaml")` exists
       - If not, return `Ok(None)`
       - If yes, read file contents, deserialize with `serde_saphyr::from_str::<AgentConfig>()`
       - On deserialization error, return `AgentError::InvalidConfig` with the agent name and error reason string
       - Return `Ok(Some(config))`

       `discover_agents(agents_dir: &Path) -> miette::Result<Vec<AgentDef>>`:
       - Read `agents_dir` with `std::fs::read_dir`, wrap IoError
       - For each entry that is a directory:
         - Extract directory name as string
         - Call `validate_agent_name`
         - Check for `IDENTITY.md` — if missing, `tracing::warn!` and skip
         - Check for `policy.yaml` — if missing but IDENTITY.md exists, return `AgentError::MissingRequiredFile`
         - Call `parse_agent_config` for optional `agent.yaml`
         - Check for `.mcp.json` existence, store as `Option<PathBuf>`
         - Check for optional OpenClaw files (SOUL.md, USER.md, MEMORY.md, AGENTS.md, TOOLS.md, BOOTSTRAP.md, HEARTBEAT.md), store as `Option<PathBuf>`
         - Construct `AgentDef` and push to vec
       - Sort agents by name for deterministic ordering
       - Return collected agents
