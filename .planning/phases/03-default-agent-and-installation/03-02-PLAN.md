  <action>
Create `install.sh` at repository root. This is a standalone bash script, not Rust code.

Per D-16: Downloads pre-built binaries from GitHub Releases with platform detection:
- `uname -s` -> linux or darwin
- `uname -m` -> x86_64/amd64 or arm64/aarch64
- Construct download URL: `https://github.com/user/rightclaw/releases/latest/download/rightclaw-{arch}-{platform}`
- Download to `~/.local/bin/rightclaw` (create dir if needed)
- Add `~/.local/bin` to PATH for current session if not already there

Per D-17: Check for existing installations before installing each dependency:
- `command -v process-compose` -- skip if found
- `command -v openshell` -- skip if found
- Print "already installed" messages for each found tool

Per D-18: Use official install mechanisms:
- process-compose: `curl -LsSf https://raw.githubusercontent.com/F1bonacc1/process-compose/main/scripts/get-pc.sh | sh -s -- -b ~/.local/bin`
- OpenShell: `curl -LsSf https://raw.githubusercontent.com/NVIDIA/OpenShell/main/install.sh | OPENSHELL_VERSION=v0.0.13 sh`

Per D-19: After all installs complete, run `rightclaw init` using FULL PATH (`~/.local/bin/rightclaw init`) to avoid Pitfall 6 (PATH not refreshed in current shell). This triggers Telegram token prompt interactively.

Per D-20: Run `rightclaw doctor` at the end using full path to verify everything.

Script structure:
```bash
#!/usr/bin/env bash
set -euo pipefail

# Color output helpers
# Platform detection (OS + arch)
# Install directory setup (~/.local/bin)
# Step 1: Install rightclaw binary (from GitHub Releases, with cargo install fallback)
# Step 2: Install process-compose (skip if present)
# Step 3: Install OpenShell (skip if present)
# Step 4: Run rightclaw init
# Step 5: Run rightclaw doctor
# Print success message with next steps
```

Include a `cargo install` fallback for rightclaw if GitHub Releases download fails (per open question 3 in research -- no release pipeline yet). The fallback runs `cargo install --path crates/rightclaw-cli` if the repo is cloned, or `cargo install rightclaw-cli` from crates.io.

Also check for `bun` and print a warning if Telegram token is provided during init but bun is not installed (Telegram plugin requires Bun runtime per Pitfall 2).

Mark the script executable: file should have `#!/usr/bin/env bash` shebang.
