**Step 2 — Implement BotProcessAgent struct in process_compose.rs:**
```rust
#[derive(Debug, Serialize)]
struct BotProcessAgent {
    name: String,          // "<agent-name>-bot" (the PC process key prefix, template appends -bot)
    agent_name: String,    // agent.name (used in --agent arg)
    exe_path: String,      // exe_path.display().to_string()
    working_dir: String,   // agent.path.display().to_string()
    token_inline: Option<String>,  // RC_TELEGRAM_TOKEN value (if inline token)
    token_file: Option<String>,    // RC_TELEGRAM_TOKEN_FILE value (if file, absolute path)
    restart_policy: String,
    backoff_seconds: u32,
    max_restarts: u32,
}
```

Token precedence (per Pitfall 6 and RESEARCH.md Pattern 6):
- `telegram_token_file` set → `token_file = Some(agent.path.join(rel_path).display().to_string())`, `token_inline = None`
- `telegram_token` set → `token_inline = Some(token)`, `token_file = None`

Filter: only agents where `config.telegram_token.is_some() || config.telegram_token_file.is_some()`.

New function signature: `pub fn generate_process_compose(agents: &[AgentDef], exe_path: &Path) -> miette::Result<String>`

Drop `run_dir` parameter entirely. `restart_policy_str` helper stays unchanged.

**Step 3 — Rewrite templates/process-compose.yaml.j2:**
```
# Generated by rightclaw -- do not edit
version: "0.5"
is_strict: true

processes:
{% for agent in agents %}
  {{ agent.name }}-bot:
    command: "{{ agent.exe_path }} bot --agent {{ agent.agent_name }}"
    working_dir: "{{ agent.working_dir }}"
    environment:
      - RC_AGENT_DIR={{ agent.working_dir }}
      - RC_AGENT_NAME={{ agent.agent_name }}
{% if agent.token_inline %}
      - RC_TELEGRAM_TOKEN={{ agent.token_inline }}
{% else %}
      - RC_TELEGRAM_TOKEN_FILE={{ agent.token_file }}
{% endif %}
    availability:
      restart: "{{ agent.restart_policy }}"
      backoff_seconds: {{ agent.backoff_seconds }}
      max_restarts: {{ agent.max_restarts }}
    shutdown:
      signal: 15
      timeout_seconds: 30
{% endfor %}
```

No `is_interactive` field anywhere. Note: `{{ agent.name }}-bot` in the template — the struct field `name` = `agent.name` (not pre-suffixed). The `-bot` suffix is in the template.
