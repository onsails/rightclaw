<objective>
TDD fix for startup_prompt in shell_wrapper.rs: write two failing regression tests first, then replace the
constant so rightcron bootstraps inline on the main thread instead of delegating to a background Agent.

Purpose: The root cause of the Telegram/CronCreate bug (Phase 20 diagnosis) is that the background Agent
dispatched by startup_prompt cannot see main-thread-only tools like CronCreate. Removing the Agent tool
delegation means bootstrap runs in the main thread where CronCreate is available.

Output: Two new regression tests + one-line constant change at shell_wrapper.rs line 54.
</objective>
