<objective>
Restructure skills/cronsync/SKILL.md to add a CRITICAL guard against Agent tool delegation and split the
reconciliation algorithm into Step A (CHECK: read-only diff computation) and Step B (RECONCILE: direct
CronCreate/CronDelete calls in main thread). Also annotate bootstrap step 2 with a direct-call reminder.

Purpose: Prevent the LLM from ever delegating CronCreate/CronDelete to a background Agent, which silently
fails because those tools are main-thread-only. The structural split (CHECK before RECONCILE) makes the
constraint visible and enforced by the algorithm structure itself.

cargo build --workspace to embed the new skill content into the binary via include_str!.
