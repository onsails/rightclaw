<objective>
Replace the proprietary `.credentials.json` key-derivation credential storage with direct `.mcp.json` header injection.

Purpose: CC cannot find tokens written via `mcp_oauth_key()` because it uses a different key derivation internally. By writing `Authorization: Bearer <token>` directly into `.mcp.json` headers, CC passes the header as-is to MCP servers â€” no credential lookup needed.

Output: Rewritten `credentials.rs` with `.mcp.json`-targeting functions, rewritten `detect.rs` that checks headers instead of credentials file.
</objective>
