## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| OAuth callback → .mcp.json | Token received from provider written to local file |
| Refresh loop → .mcp.json | Refreshed token replaces old in local file |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-41-04 | Tampering | .mcp.json concurrent writes (refresh + callback) | mitigate | write_json_atomic uses NamedTempFile::persist() — atomic replace, last writer wins (acceptable for token rotation) |
| T-41-05 | Denial of Service | Removing agent restart | accept | CC uses `claude -p` per message — no persistent session means restart was a no-op anyway; removing it is correct |
</threat_model>
